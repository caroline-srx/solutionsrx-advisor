SolutionsRx® Supplement Advisor — Embed Instructions

Add the Supplement Advisor as a popup on any website in two steps.

Step 1 — Add the script tag

Paste this once, anywhere before the closing </body> tag on your page (or in your WordPress footer):

<script src="https://demo.solutionsrx.com/embed.js"></script>

Step 2 — Add a button

Place this button wherever you want it to appear on the page:

<button data-srx-advisor style="
  background: #bf0a30;
  color: #ffffff;
  border: none;
  padding: 14px 28px;
  font-size: 16px;
  font-weight: 700;
  border-radius: 12px;
  cursor: pointer;
">
  💊 Find My Supplements
</button>

Optional — Trigger from any element

Any element with data-srx-advisor will open the popup on click. Or call it from your own JavaScript:

// Open the advisor from anywhere in your code
SRXAdvisor.open();

WordPress instructions

  1. Go to Appearance → Theme File Editor (or use a plugin like Insert Headers and Footers)
  2. Paste the <script> tag into the footer section
  3. Add the button HTML to any page or post using the Custom HTML block in the editor
⚠️ Note: The popup loads demo.solutionsrx.com inside an iframe. Make sure your website is served over HTTPS — most modern sites already are.
